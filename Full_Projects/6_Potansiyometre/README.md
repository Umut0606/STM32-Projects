# 6_Potansiyometre - Potansiyometre ile PWM LED Parlaklık Kontrolü

Potansiyometreden alınan analog veri ADC ile okunur ve PWM sinyali üretilerek LED parlaklığı kontrol edilir.

## 🛠 Kullanılanlar
- STM32F407G-DISC1
- STM32CubeIDE
- Breadboard
- LED, LDR vb.

## 🔌 Bağlantılar
- ADC Girişi: PA1
- PWM Çıkışı: PD12 (TIM4 CH1)
