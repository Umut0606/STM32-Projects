# 7_LDR_Harici_LED - LDR ile Harici LED Kontrolü

LDR sensör ile ortam ışığı ölçülür ve buna göre LED açılıp kapanır.

## 🛠 Kullanılanlar
- STM32F407G-DISC1
- STM32CubeIDE
- Breadboard
- LED, LDR vb.

## 🔌 Bağlantılar
- ADC Girişi: PA1
- PWM Çıkışı: PD12
